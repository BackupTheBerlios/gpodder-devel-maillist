Hello,

Since last week there exists a podcast called "Tin Foil Hat" /
http://my.gpodder.org/podcast/49014
You can find more information on http://cafeninja.blogspot.com.

As you can read in the post for the first episode, there is something
special with this podcast. You couldn't find the shownotes on the
mentioned blog. You have to extract the shownotes from the media file. 

Because I don't want to extract the shownotes for every episode manually
I wrote a small script for gPodder. You can set the script file name in
the advanced configuration editor variable "cmd_download_complete".

The script extract the shownotes after the download and save it into the
gPodder database, so you can read the shownotes directly in gPodder
I will attach some screenshots to show you the results.

If you are interested you could find my script on github. I would like
to hear your suggestions and improvements. Of course you could fork the
project and adapt it to your needs.
http://github.com/brot/gPodder-TinFoilHat-shownotes


~ Bernd
