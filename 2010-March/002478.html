Mtp devices can handle playlists, but define tracks instead of files, which
are indentified with property likes genre, title, album artist... instead of
file name (allthought there is a filename property, but it is not mandatory
and must not be unique) . Audio files in a filesystem also have this
properties in their tags. Nevertheless, each device could define its own
fields in the view.
