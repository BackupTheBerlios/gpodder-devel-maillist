-----
2010-03-15 mygpoclient 1.2 "Lighthouse" released

This release adds Windows compatibility and includes a fix for a bug
that happened with relative imports when simplejson was not installed.
This update is recommended for all users. If you cannot upgrade to 1.2
at the moment, please make sure to install python-simplejson so that the
relative import bug in version 1.1 will not affect you.

Please note that version 1.1 was generating wrong URLs in Windows, as it
was using os.path.join to construct URLs, which does not work under
Windows, because the URLs contained backslashes. This has been fixed in
version 1.2. The version of mygpoclient that is bundled with gPodder 2.3
for Windows already has this fix included as a hotfix. Future versions
will simply ship with 1.2 and later.
