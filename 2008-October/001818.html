Hello there!

I've just branched 0.13.1 from 0.13.0 and cherry-picked all bug fixes 
from the current development branch to it. Please test this branch 
(normal "git clone" and then "git checkout gpodder-0.13.1") and report 
any problems found. This will get to be what 0.13.1 is when we release 
it (in a week or so if everything works out as expected).

Also, if you are a translator, please send me updated translations for 
gPodder now (please "make messages" before sending the translation file, 
so you can be sure to update the latest strings for 0.13.1).

If you have some bugfix patch sitting around somewhere that you think 
should get into 0.13.1, please get in touch soon!

As with the 0.12.x release series, the 0.13.1 release will only contain 
bugfixes and usability improvements that have been found since the 
0.13.0 release. Feature patches can live happily in the development 
branch until 0.14.0 comes out.

But before that, we want to make 0.13.x as solid as possible - please 
help testing:

   See http://wiki.gpodder.org/wiki/Running_gPodder_from_Git for how to
   get the latest development version and then switch to the release
   branch with the command "git checkout gpodder-0.13.1" and test it
   with "make test".


Thanks,
Thomas
