I am happy with just setting the genre ID3 tag to "Podcast" for all my
podcasts then on my player (Sandisk Sansa c240) I can play by genre. I can't
select the order, but that is not important to me. I was using EasyTag to
change the tag after download, but now I've actually put some code into
gPodder that will do it for me. I added a check box in preferences in the
File Metadata section with the lable 'Force genre tag to "Pogcast"' if that
(and update tags...)is checked, it will do just that, force the genre tag to
Podacast.

I'm new to gPodder and new to Python/Gtk/Glade and I've been having trouble
making a patch. I also wanted to read through some archives before
submitting anything.
