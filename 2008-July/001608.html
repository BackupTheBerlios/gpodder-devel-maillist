Hello!

There's been little news about a new gPodder release lately, mostly
because we've been working hard to get the SQLite patch in good shape
and add some other goodies to SVN trunk.

We'll be releasing gPodder 0.12.0 on or around July 15th, 2008. I've
updated the Roadmap on the Wiki accordingly.

No SVN branch today, as we're still waiting for some patches to land on
trunk. After that, we will create a proper branch for 0.12.0 in our
Subversion repository.

Please help testing and hang around on #gpodder on FreeNode if you want
to help make 0.12.0 one of the best gPodder releases so far :)
