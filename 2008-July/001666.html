I've been asking that question some months ago, but as we're developing
more de-centralized and with more developers now, nikosapi, Justin
Forest and I have been talking on IRC about switching to Git today.

Before we start switching all over, I'd like to have some input from all
interested developers and users - give us some pros and cons!

change, everything else would stay basically the same. For users writing
patches, we'd be able to make it easier to merge patches and do
experimental branches.
