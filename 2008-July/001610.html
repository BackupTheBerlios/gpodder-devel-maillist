Hello!

We need some people to test a patch which will present the user with a message 
if they don't have sufficient free space on their device to hold the episodes 
which are about to be copied. It would be nice to have this feature in the 
next release but it needs to be tested first.

You can find the patch here: http://bugs.gpodder.org/attachment.cgi?id=109 and 
the original bug report here: http://bugs.gpodder.org/show_bug.cgi?id=128

If you're not keen on filling your device to the brim you can try the method 
described in comment #5 to set up a "fake" device which is constrained to 
~61MiB.

Thank you for your help,

nick

P.S. Please let us know what OS you're using and whether it's the 32 or 64 bit 
version. (I'm having some issues on Fedora 9 x64 that seem to be related to a 
problem in libxcb)
