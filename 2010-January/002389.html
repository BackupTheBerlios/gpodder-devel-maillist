Hello!

I've relased the first public version of the "mygpoclient" library
today. The mygpoclient library provides a Pythonic interface to the
my.gpodder.org API. With the library it's possible to talk to the Simple
and Advanced APIs of my.gpodder.org. In addition, accessing public
information (toplist and searching at the moment) is also possible.

The library is licensed under the GPL version 3 and will become a new
dependency for gPodder in the upcoming release of gPodder. If you are a
packager, please start packaging mygpoclient now so that you can satisfy
this dependency in the next package version of gPodder.

Here are some important URLs related to this release:

Homepage (+Download): http://thpinfo.com/2010/mygpoclient/
GitWeb: http://repo.or.cz/w/mygpoclient.git
API docs: http://wiki.gpodder.org/wiki/Web_Services/API
Examples: http://wiki.gpodder.org/wiki/Web_Services/API/Reference

I'm looking forward to someone implementing my.gpodder.org integrating
for existing applications or even creating a standalone companion
utility for managing my.gpodder.org subscriptions on the Desktop.
