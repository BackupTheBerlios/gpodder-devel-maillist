Hi Chris

If you run gPodder from source by entering 'make test' in a terminal at
the top level source directory, debug information will be output to the
terminal window that may help you find out what's going on.  Further
details on doing this can be found at the following wikipage:

http://gpodderwiki.jottit.com/running-from-svn

I hope this helps.  Let us know how you get on.
