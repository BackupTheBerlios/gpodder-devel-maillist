Hello!

We've been fixing bugs left and right in the last few weeks, and now
have a polished version of gPodder 2.2 in the git repository. Before we
are going to add more features (and improve existing ones), I want to do
another "bugfix" release - that's gPodder 2.3.

Now, it's time to prepare for the next release. If you are using the Git
version, please "git pull" the latest changes and test them in everyday
use. If you have a N800, N810 or N900, you can also test the Git version
by cloning the repository and running gPodder using "bin/gpodder --maemo
--verbose" (for N800+N810) or "bin/gpodder --fremantle --verbose" (for
N900) on your device.

If you are translating gPodder into another language, please update your
translation, preferably via Transifex:

http://www.transifex.net/projects/p/gpodder/c/master/

Postponed features and patches will be worked on for the release after
this upcoming one. Again, let's make 2.3 a polished bugfix release :)


Thanks for your help!

Thomas
