Hello!

I've just committed the first version of a download queue for gPodder.
You can enable and configure it in the preferences dialog. Currently, a
restart is required for the changes to take effect, but otherwise it
should run smoothly.

Combined with the rate limiting functionality, we now have a nice
infrastructure for limiting downloads so gPodder doesn't interfere with
web browsing and interactive ssh sessions on lower-bandwidth
connections.

As always, feedback to this feature is welcome :)


Enjoy,
