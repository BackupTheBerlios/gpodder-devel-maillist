On Sun, 2007-05-27 at 16:34 +0200, Tim Michelsen wrote:
> I tried to change the default download directory ~/gpodder-downloads
> to /media/share/podcasts on a vfat drive.

If moving the download folder doesn't work for you, it might be related
to not having enough space left on the destination device or some other
permissions problem. You can always move the folder manually:

1. Close gPodder
2. Move the download directory to the new destination
3. Change the "download_dir" setting in ~/.config/gpodder/gpodder.conf
4. Restart gPodder

gPodder should now use the new download directory location.

> * How do I import a local OPML file with my Podcast list from my web 
> aggregator?

Enter the web location of your OPML file in the preferences dialog and
click on "Podcasts" -> "Channel list" -> "Import from web". Then, add
the channels you want to add to your gPodder channel list.
