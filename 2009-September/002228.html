You can use "gpodder --maemo --verbose | tee gpodder.log" to run gPodder
for Maemo in xterm and log the output to the file "gpodder.log" in
addition
to displaying error messages directly in the X Terminal.

A new version of gPodder will be released soon which fixes some bugs
that
have been discovered since the release of gPodder 0.17.0.
