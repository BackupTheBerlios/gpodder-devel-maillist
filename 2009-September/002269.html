You wrote:
> I've been using the 2.0 version of gpodder on Maemo and noticed that
> the trash can doesn't appear next to a deleted episode anymore. Is it
> possible to re-enable that? I used it to know which episode to
> download next but now I don't know which ones I listened and which
> ones I didn't.

Actually, it does appear, but deleted episodes are hidden by default.
I think the "view" menu should be made more discoverable, as I had
another user having the same problem:

http://talk.maemo.org/showpost.php?p=328399&postcount=343

So, in short, "View" -> "Show deleted episodes" will un-hide ("show")
the deleted episodes and display them with their trash can icons.

The same is true for the Desktop version, as well.
