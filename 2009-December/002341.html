Hello!

If you have not been reading the news in the last few days, you might
have missed that we have now released the Windows version of gPodder
2.1, which includes support for ,,portable mode``, so that you can run
gPodder off your MP3 player in USB mass storage mode, and downloads and
settings will be saved directly in the gPodder folder, so that you can
roam around freely with your device and download podcasts.

In related news, gPodder 2.1 has gone through Maemo's Extras-Testing
process and is now available in Maemo.org Extras for the N900. The Maemo
4 version for the N800 and N810 has been available for some weeks, as it
does not have to go through Testing in these versions.


Thanks,
