Lucas (MaskMaster) & me will continue to work on gpodder.net Features
in Amarok:

- -) Synchronize more Status Information (also the downloaded, played
and paused) with gpodder.net
- -) Synchronize local Subscriptions with gpodder.net on first start of
the service
- -) Search Podcasts via gpodder.net in the Service View
- -) on set up of the service, if the "amarok" device was not found,
display an UI to synchronize it with existing devices

If anyone has wishes what to see in Amarok 2.6 related to gpodder.net
just write to the Mailinglist ;)
