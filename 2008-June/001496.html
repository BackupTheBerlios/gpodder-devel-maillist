gPodder's "opml" module to load the OPML file and channels, then for
each channel do the following:

1) create an empty dict
2) for each episode:
   *) call the (modified) local_filename() method, save the return value
   *) if the return value is not in the dict, add it and continue
   *) if the return value is in the dict, there has been a collision

functions.
