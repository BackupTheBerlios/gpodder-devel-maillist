On Wed, 2008-06-04 at 10:54 -0400, Chris Witham wrote:
> I don't have much on the iPod itself, so
> I'm not opposed to recreating the DB if that will fix it, but I'm not
> sure how to do that without using iTunes.  (I avoid using iTunes if at
> all possible.)
> 
> Does anyone have any suggestions on how to correct this problem?

Delete everything on the iPod (including the hidden directory
iPod_Control or how it's called) and then open up "gtkpod" and
"initialize" the ipod database (in the menu) and then copy at least one
audio file on it (this is needed for gtkpod to write the first version
of the iTunesDB or something like that - at least people have complained
about not being able to use the iPod with gPodder when having a
completely fresh iPod, so just copy one song onto it with gtkpod).

HTH
