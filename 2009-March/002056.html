Hello,

I downloaded and tested your changes and I have to say, that there is a
problem now for me in the playlist and sync functionality.

In the "old" version there was the "mp3-player-folder" which is missing
in your new version.
As far as I can see you use the device mountpoint instead of this
setting. But this could be a different location 

I am using a Meizu filebased player with the following file-structure:

mountpoint: /media/MEIZU-M6

directories:
MUSIC
PICTURES
PLAYLISTS
RECORD
SYSTEM
VIDEO

For my podcasts I created the folder "/media/MEIZU-M6/MUSIC/Podcast".
This was my value for the "mp3-player-folder" setting.
So the playlist-function only searches in this folder and their
subfolders. Also the sync process takes care of this location.

But now the file-sync and playlist function only use the folder
/media/MEIZU-M6/MUSIC.
So all my podcasts was synchronized to the device on another place and
the playlist-function reads all folders under MUSIC and listed all
content and not only my podcast?!

What could we do to solve this issue?
