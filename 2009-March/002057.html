> I downloaded and tested your changes and I have to say, that there is a
> problem now for me in the playlist and sync functionality.
>
> In the "old" version there was the "mp3-player-folder" which is missing
> in your new version.
> As far as I can see you use the device mountpoint instead of this
> setting.

Not exactly. There's a list of folders reported by freedesktop, for
Meizu M6 it's MUSIC/ and RECORD/. The current gPodder code uses the
first entry from that list.  You can override this by creating a file
called /media/MEIZU-M6/gpodder.conf with the following contents:

[default]
audio_folders = MUSIC/Podcast

Later there will be some GUI to tune such things.
