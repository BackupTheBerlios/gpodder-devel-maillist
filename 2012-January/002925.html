On Sat, Jan 07, 2012 at 12:06:10PM -0800, simplicitasetmusica39.amabel at recursor.net wrote:
> Hi. We run Linux Mint 11. How do we install the latest version of Gpodder,
> currently 3.0.2? We downloaded the tar.gz, but don't know how to continue.

If you really want gPodder 3.0.2 (device sync is not yet available in
3.x, see [1] for more info) you can simply use the .deb from Debian
Experimental[2] usually.

Downloading the .tar.gz and building it yourself is not recommended,
especially if you don't really know what you are doing, so the .deb way
is the safest way. Or simply wait until we package 3.x for the Ubuntu
PPA and also let it go into Debian unstable (this will happen when 3.x
regains device sync functionality).

HTH.
Thomas

[1] http://blog.gpodder.org/2011/11/device-sync-in-3x-brainstorming-for.html
[2] http://packages.debian.org/experimental/gpodder
