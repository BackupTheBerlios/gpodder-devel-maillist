I have not yet tried Feisty Fawn, as I'm happily using Debian
testing/unstable here. You should be able to build a gPodder package for
Ubuntu by using the Debian sources:

mkdir ~/tmp-build && cd ~/tmp-build
dget
http://mentors.debian.net/debian/pool/main/g/gpodder/gpodder_0.9.1-1.dsc
dpkg-source -x gpodder_0.9.1-1.dsc
cd gpodder-0.9.1
debuild
cd ..
dpkg -i gpodder_*.deb

You will have to install some build dependencies if it tells you to.
