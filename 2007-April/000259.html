Hi Darren,

you cannot simply install random packages from the net, that were not
built explicitly for your distribution. From your question below I
read that you are using Ubuntu, while these packages are for Debian.

> Any further hints/guidance you can offer? Will this problem likely go
> away with the release of Feisty Fawn?

If gpodder cannot load the python modules, this could mean that
gpodder and your packages were built for different versions of python.
Python modules for different versions of Python are in different
paths, e.g. /usr/lib/python2.4/site-packages vs.
/usr/lib/python2.5/site-packages on Mandriva 2007.0 vs. Mandriva
2007.1.
