The latest version of gPodder is 0.9.1, released in early April. You're
using 0.9.0, please upgrade to the latest version, which includes
bugfixes for the problems you mentioned (don't know about the VPN thing,
though .. this might need more investigation).

Because you're using a distribution package, I suggest you file a bug
against Gentoo's bug tracker and state that a new upstream version is
available that fixes the bugs mentioned above (including some more, see
the gPodder website for full information).

Alternatively, you could install gPodder from the 0.9.1 source tarball.

Tell me if the VPN and empty channel list problems go away with 0.9.1.
If not, we'll have a look at it.
