> You have selected "iPod" as type of device and not "Filesystem-based MP3
> player"? iPods use a file called iTunesDB that hold metadata information
> about the files on the device, and this file needs to be updated for the
> iPod to display the files in its menus.
> 
> Select type of device: iPod in your preferences dialog. You need to
> install python-gpod and at least one of python-pymad or python-eyed3.
> 
> Does this solve your problem?
