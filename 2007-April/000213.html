You need to uninstall all gpod-related packages for Ubuntu, then build
and install libgpod[1] and then install its Python bindings.

[1] http://www.gtkpod.org/libgpod.html

The Python bindings can usually be found in "bindings/python" in the
libgpod source directory. Run "make" and then "make install" - this
should install the right version of the bindings on your system.
