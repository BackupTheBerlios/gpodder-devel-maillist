Aravind Seshadri wrote:
> Does anyone know if it is possible to do synchronization using the
> command line.

If you mean "MP3 player sync" or "iPod sync", then no, not at the
moment. Need volunteers or donate some free time ;)

For simple updating and downloading of feeds/episodes, have a look at
"gpodder --help":

usage: gpodder [options]

gPodder enables you to subscribe to RSS feeds and download
podcast episodes from these feeds. gPodder can operate in
GUI mode and in CLI mode. Downloaded podcasts can either
be synchronized to portable MP3 players (including iPods)
or played back on the user's desktop.

options:
  --version             show program's version number and exit
  -h, --help            show this help message and exit
  -v, --verbose         Print debugging output to stdout
  -t, --local           Run local version in current directory
  -l, --list            List all channel subscriptions
  -r, --run             Update channel list, download new podcasts
  -u, --update          Update channel list and exit
  -a URL, --add=URL     Subscribe to channel from URL
  -d URL, --delete=URL  Delete channel specified by URL
