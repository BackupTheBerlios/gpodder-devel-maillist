> Hello Thomas and list! I use to use SLED and was not able to use
> gpodder 0.10.3 so i installed opensuse 10.3 and now am trying to
> get gpodder 10.3 installed. When i run make install, i get this:
--- 8< ---------
> make[1]: Leaving directory `/root/Desktop/gpodder-0.10.3/data/po'
> python setup.py install --root=/ --prefix=/usr
> running install
> error: invalid Python installation: unable to open /usr/lib/python2.5/config/Makefile (No such file or directory)
> make: *** [install] Error 1
--- 8< ---------
> Could you instruct me on how to get gpodder 0.10.3 installed on
> opensuse 10.3? Thanks for all your help

Try installing the Python development package, it should be named 
something like python-dev or python-devel or python-2.5-devel. Maybe you 
also need python setuptools or distutils.. These packages should be 
provided by OpenSuSE.
