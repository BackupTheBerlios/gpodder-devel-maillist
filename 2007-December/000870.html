Hello, Sean!

Sean Rima wrote:
> When I sync podcasts, should Gpodder remove from the ipod played
> podcasts?  I cannot see how to do it if it does.

Currently, this is not implemented. The only thing you can do from the
gPodder GUI is delete _all_ podcasts from the iPod. I'm currently
working on new sync code that can delete podcasts individually, just
like the current "remove old episodes" (ctrl+k) dialog.

Until this functionality is implemented in gPodder, you can delete these
podcasts with gtkpod.
