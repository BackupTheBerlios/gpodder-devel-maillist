Hello, List!

I've just released gPodder 0.9.3 as a bugfix release to the 0.9.2
release. Here's the official announcement:

This is a bugfix release adding all fixes that have accumulated in our
code repository since the 0.9.2 release. This release improves track
length detection for iPod sync, updated translations from launchpad.net
(Rosetta) and fixes several important bugs.

As always, distribution packages will be available from the usual
locations in the upcoming days. You can download the source tarball for
gPodder 0.9.3 now from Berlios' download servers.


Changes since gPodder 0.9.2
      * Bugfix: Untranslated strings in the iPod sync GUI fixed
      * Bugfix: Make "Update feed cache on startup" work correctly
        (thanks to Götz Waschk and Stephan Buys)
      * Bugfix: use urllib.unquote() for URLs containing the "%"
        character (thanks to Thomas Matthijs)
      * Bugfix: Fix logger import error in libopmlreader (thanks to
        Nick)
      * Prefer mplayer for track length detection to fix a bug for
        enhanced AAC files (thanks to Götz Waschk)
      * Translations updated from launchpad.net (Rosetta)
      * 
Thanks
      * Fixes and bug reports: Götz Waschk, Stephan Buys, Thomas
        Matthijs and Nick (nikosapi)
      * Donations: Jessica Henline
