Ah! gPodder is trying to update the OGG file's metadata using
vorbiscomment (i.e. assuming every OGG file is a vorbis file).

Look for a fix for this bug soon :) In the meantime, you can circumvent
this problem by disabling the "Update tags on audio files after
download" option in the "Extras" tab of gPodder's preferences window.
