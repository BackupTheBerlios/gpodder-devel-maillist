Hello!

Launchpad.net (Canonical's web service for open source software
development) has accepted the translation template for gPodder.

This means we can now translate gPodder using an easy-to-use web
interface, here's the URL:

https://translations.launchpad.net/gpodder/trunk/+pots/gpodder/

I'll try to keep the translation templates recent, so that if anyone
decides to update an existing translation or translate gPodder into a
new language, this can be done easily on launchpad :)


Thanks,
