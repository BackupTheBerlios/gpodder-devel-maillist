Hello,

This e-mail is just a round up of my favorite bug and feature requests
so people can read about and maybe vote for them. There are  probably
more important bugs out there that I can't really see or don't affect
my usage. These are just my personal favorites. If you have other
favorite bugs you can talk about them and others might vote.

Sometimes Gpodder can stop responding for a brief (or sometimes long)
period of time. This usually happens when you synchronize podcasts
with your iPod and more severely when you import an opml file. I
couldn't find a bug report for this so I opened a new one one. You can
vote here:
http://bugs.gpodder.org/show_bug.cgi?id=245

Some actions the user does can cause multiple message dialogs to pop
up. For example when you import an opml file which contains podcasts
you already have, gpodder will show a message dialog for every
instance of this fact. It happened to me on other instances but I
can't remember right now. I think it was also when syncing with the
iPod. If you have experienced this problem or think this should be
addressed, you can vote for it here:
Collect multiple error messages in one MessageDialog.
http://bugs.gpodder.org/show_bug.cgi?id=173

Gpodder does not timeout downloads when network connection is lost.
Vote for it here:
http://bugs.gpodder.org/show_bug.cgi?id=174

If you want gpodder to retry downloads after failure, vote for this request:
http://bugs.gpodder.org/show_bug.cgi?id=137

If you would like to have a search feature in gpodder please vote for
this request:
http://bugs.gpodder.org/show_bug.cgi?id=13

Best regards,
Silvio Sisto
