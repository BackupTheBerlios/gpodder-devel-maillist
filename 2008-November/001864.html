> Hi there!
>
> I've had a discussion recently about the URL entry box in gPodder (the
> one inside the podcast list). You can currently disable it with the menu
> items "View" -> "[x] Show URL entry box".
>
> If the URL entry box is not shown, we have a separate dialog (Ctrl+L or
> use the menu) to enter the URL.
>
> I am thinking of removing the URL entry box completely and only having
> the separate dialog (still quickly accessible via Ctrl+L) as the only
> text-based means of adding podcasts to gPodder.
>
> Are there any objections against removing the URL entry box? Are there
> hardcore users around that subscribe to a lot of podcasts and find the
> URL entry box handy for that?
