I send you a first patch:

Description:
* When "auto-update feeds" and "auto-dowload new episodes" are set and 
gPodder is iconified, an ugly message box pops up every x minutes to say 
"no new episode found". Worse, If you let your computer alone for a few 
hours, you'll have dozen of this messages to close when you'll come back.
This patch should fix it.

* when gPodder is iconified and send a message (for exemple when gPodder 
is not able to read channel), a message box pops up on the screen. This 
patch send a notification instead.

Jérôme
