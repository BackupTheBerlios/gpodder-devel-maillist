Hello, Ben!

On Thu, 2008-01-24 at 08:41 -0500, Ben Wilson wrote:
> I am looking for a way to append newly downloaded podcasts to existing
> playlists.
> 
> Is this something already in gpodder? I see the "playlist" setting in a
> channels sync config, but it doesn't see to be doing what I want.

Currently not, the "Playlist" setting in the channels sync config is
only for syncing to iPods and will be removed soon (because syncing to
playlists in iPods is not necessary anymore - we now support the special
"Podcasts" playlist).

For the moment, you can simply create the playlist by doing:

cd /path/to/your/mp3/player/podcasts/
find . -name '*.mp3' > podcasts.m3u


Enjoy!
Thomas
