Hello!

I've just released gPodder 0.10.4. You can grab the tarball from the
project website now or wait until it gets packaged in your distribution.

Thanks to all of you who contributed to this new release of gPodder!

Release notes: http://gpodder.berlios.de/news.html
Download: http://gpodder.berlios.de/downloads.html

The release notes follow...

----------------------
After the early December release, we had enough time to prepare this
special release: gPodder 0.10.4 is here, and it shines with its updated
translations, some minor feature enhancements and many bug fixes that
make this release a very stable one. Of course, this also means that we
have not included all the cool features that you can already find in our
Subversion repository, but it means you get a stable, usable podcast
downloader. The new features currently in development will be stabilized
until late February for the next release.

In this release cycle, not only did we enhance the gPodder Wiki and had
fun trying to get the iPod Touch working (thanks to Pieter De Decker),
we also got a brand new, Bugzilla-based bug tracker that we plan to use
for keeping track of bug reports and feature requests - thanks to Paul
Rudkin for taking the time to set everything up and for hosting the
Bugzilla installation. I am also very grateful for the two books I got
sent via my Amazon Wishlist - it is a nice motivation for doing some
more work on gPodder. Last but not least, I want to thank all the people
that have contributed to this release, they are listed below (and I hope
I didn't forget anybody).

You can grab the new release tarball from the downloads page or wait
until the new version makes it into your distribution's package
repositories. If you have any questions or problems, please see our
FAQs, our Wiki or the Mailing list. Thanks.


Feature enhancements
      * Menu item for gPodder's Bugzilla in Help menu
      * Add option to delete episodes in gPodder after synchronization
        (patch from Jérôme Chabod)
      * Add option to keep downloaded episodes when removing channels
        (suggested by Paul Rudkin)
      * Better username/password extraction from URLs (patch by Nick,
        reported by Shane Donohoe)
      * Migrate channels.xml to channels.opml when upgrading from 0.9.4
        and before (reported by Tim Michelsen)
      * Under the hood: download completion notifications, improve
        logging
Bug fixes
      * Do not synchronize played state from PC to iPod when marking
        episodes as played on sync in gPodder
      * Remove invalid channel cover images when loading them fails
      * Make sure the feed update dialog gets closed when clicking on
        "Cancel"
      * Create BitTorrent download folder on startup when needed (avoids
        a warning)
      * Native system language encoding patch for gdfs (by Leonid
        Ponomarev)
      * Fallback support for pycairo versions less than 1.1.6 (reported
        by Chris Arnold)
      * Show episodes that disappeared from the RSS feed in "Delete old
        episodes" dialog (reported by Pieter De Decker)
      * On-the-fly patching of the feedparser module to fix a bug there
        (reported by Clark Burbidge)
      * Use os.rename() to make channel list saving atomic (reported by
        Jens Thiele)
Translations
      * Updated: Dutch (Pieter De Decker), Swedish (Anders Kvist),
        Russian (Hex), French (Jérôme Chabod) and German
      * Added: Czech (Ondrej Vesely)
Thanks
      * Anders Kvist: Swedish translation
      * Chris Arnold: pycairo bug report
      * Clark Burbidge: Feedparser bug report
      * Hex: Russian translation
      * Jens Thiele: Channel list saving bug report and help
      * Jérôme Chabod: Delete episodes after sync patch, French
        translation
      * Leonid Ponomarev: Patch for gdfs
      * Nick (nikosapi): Better username/password extraction patch
      * Ondrej Vesely: New Czech translation
      * Paul Rudkin: Keep downloaded episodes feature request, Bugzilla
        setup and hosting
      * Pieter De Decker: "Delete old episodes" bug report, Dutch
        translation, iPod Touch testing
      * Shane Donohoe: Better username/password extraction bug report
      * Tim Michelsen: channels.xml to channels.opml migration feature
        request
      * Clark Burbidge sent me a nice book via the Amazon Wishlist -
        thanks!
      * Paul Rudkin also sent me a nice book via the Amazon Wishlist -
        thanks!
----------------------

Enjoy,
