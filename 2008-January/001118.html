I don't like the way the feed update is visualized now, anyway. Maybe
for the moment we should make the automatic updates only active when
gPodder is minimized (and then of course without notifications) and do
not do automatic updates when gPodder is visible.

After we fix the feed cache updating (something I've planned for the
April 08 release), we can make the automatic update also active while
the gPodder window is visible.

What do you think about my suggestion?

subscribe (CC) to the bug there.
