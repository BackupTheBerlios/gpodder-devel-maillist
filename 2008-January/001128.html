Hello!

I've just merged my private branch for the sync module with the SVN
trunk, the changes can be fetched from the repository now.

Highlights:
 * CLI-based device synchronization (gpodder --sync)
 * Delete episodes from iPods and MP3 players using the Episode selector
 * Easy framework to add new sync methods (MTP devices, ...)

Please have a look and stress-test the code to see if there are any
problems with syncing to your special device.

The functionality to select music playlists (for iPods) has been
removed, because it was a bad idea to customize the playlist to sync,
and it proved to be a problem when looking for all podcasts that should
be deleted from the iPod. The "playlist name" configuration option is
still available on a per-channel basis, because Ben Wilson said he uses
it to generate M3Us from podcasts. We might use the playlist name
configuration option in gPodder, too, to add M3U functionality to the
MP3 player sync code.


Enjoy,
Thomas
