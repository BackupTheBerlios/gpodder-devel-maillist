Loving the systray, but I have a few problems.

When gpodder is hidden to the system tray, the verbosity of everything
should be a lot lower. It is nice to have it update automatically and
check for episodes every so often. It is not nice having pop up windows,
such as the ones confirming which episodes are to be downloaded and
errors like not being able to load a channel feed. When the gpodder
window is hidden, these messages should also be hid, or shown as
libnotify notifications. It is pretty annoying having the window I am
working on lose focus due to a new pop up window from gpodder.

I suggest that the new episodes libnotify notification should get an
extra button if possible. This new button should allow the user to
choose which episodes are downloaded, with the normal download button
downloading all new episodes.

Thank you.
