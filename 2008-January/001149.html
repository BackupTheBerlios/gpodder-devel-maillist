Hello!

I've been drafting a roadmap for gPodder's development for some days now
and I think it's now ready to be discussed here on the mailing list, the
roadmap can be found in the Wiki:

http://gpodderwiki.jottit.com/roadmap

Of course, all this is not set in stone and depends on the contributions
and help that the project gets (that is, from you as contributors ;).

Jerome is currently working on finishing up the system tray icon and I'm
currently working on the new episode list view (title and description
merged). After these items have been done, the only thing that is
missing for the February release is the expert configuration dialog.

The 0.11.0 stabilization period will start in a bit more than one week,
after which no new features (only the ones listed in the roadmap) will
be added to the release branch.

I hope this gives a rough overview of what's planned and where we are at
the moment.


Thomas
