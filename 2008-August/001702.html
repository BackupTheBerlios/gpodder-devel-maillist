There's already some kind of post-download hook, see this link:

http://repo.or.cz/w/gpodder.git?a=blob;f=src/gpodder/download.py#l185

The script set in the advanced configuration editor variable  
"cmd_download_complete" can be used to convert the file via the  
GPODDER_EPISODE_FILENAME environment variable.

Please have a look if this enables you to automatically convert the  
file after a download and report back if there is still something to  
do for better integration. We should document the procedure of using  
the script and the "on_download_complete" script hook in the Wiki -  
help appreciated :)
