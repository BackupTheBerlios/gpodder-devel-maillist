Hello!

Based on a suggestion by Paul Elliot in early October[1], and thanks to
the comfort the new config manager brings us, I've now implemented a new
option for FS-based MP3 player synchronizing:

 * Create a subfolder for each channel (default = Yes)

If you unset this option in the preferences dialog, gPodder will sync
all episodes directly in the folder specified by "Sync to folder:",
without any subdirectories. This is useful for users with MP3 players
that do not support subfolders or for users with many channels and only
a few episodes per channel.

As an added bonus, the preferences dialog now hides all settings from
the "Player" tab that are not relevant for the currently selected
device, making it more intuitive and usable (hopefully ;).

The changes are now available in SVN trunk and in the next release.
