> You can also check if you're running the svn version by looking at the
> title bar of gPodder's main window. If you're running the svn version,
> you'll see something like "gPodder 0.9.3+svn20070711".
> 
> It's important that you pass the "--local" parameter to bin/gpodder if
> you have both the SVN version and Debian package installed, else it will
> use the system's copy of gPodder. I have installed the Debian package
