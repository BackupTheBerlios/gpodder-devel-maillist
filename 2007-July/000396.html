On Thu, 2007-07-12 at 09:33 -0400, Carlos Moffat wrote:
> I must be missing something here. gPodder SVN looks pretty much the
> same as the old one (no new navigation). I noticed somebody commented
> the same thing in your blog. I had the package installed from debian,
> and I uninstalled, but for some reason the GUI still doesn't show the
> changes.

Do you see the grabbing area of the paned view on the left? If so, you
have to drag it a bit to the right to "open" the new channel navigator.

You can also check if you're running the svn version by looking at the
title bar of gPodder's main window. If you're running the svn version,
you'll see something like "gPodder 0.9.3+svn20070711".

It's important that you pass the "--local" parameter to bin/gpodder if
you have both the SVN version and Debian package installed, else it will
use the system's copy of gPodder. I have installed the Debian package
myself and am happily using gPodder-svn in my home directory without
problems.

Hope that helps :)
