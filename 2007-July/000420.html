I've been playing with [K]ubuntu (7.04, not that it matters) and I noticed 
that the python-id3 package in their repos is not PyID3!
It's some old ID3 tag manipulation module that can't even do coverart. So 
attached is just a patch to remove the import code for the ID3 module seeing 
as it's not used. What people need to be using is PyID3-0.5.5.

There is a bit of a problem though, I went looking for the Py-ID3 source 
tonight and the server that was hosting it no longer has the file :(
Luckily I found a copy in an old backup of my laptop and I have it here: 
http://nikosapi.org/gpodder/PyID3-0.5.5.tar.gz

I think it's safe to say it's not being supported or developed anymore, but it 
still works so I'll host it and maybe Thomas, you can put it up on the 
project page.
