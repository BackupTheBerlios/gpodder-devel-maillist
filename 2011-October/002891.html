There are various elements in which such categories can be stored in
feeds. Unfortunately they are often used inconsistently so it would be
hard to distinguish between (high-level) categories and tags.
Therefore they are all placed into the tags field by the Feed-Service.
As far as I know we collect these tags from all commonly used elements
in the feed. If we missed something, please let me know.
