Hello!

A new release of gPodder is here, continuing our montly release series.
This release is version 2.6, and is entitled "The Staircase
Implementation". Amongst lots of bug fixes, we've integrated better
support for gpodder.net, revamped the context menu for the Desktop
version and the Maemo 4 UI and also enhanced the Maemo 5 UX.

See the source tarball or the Git repository for a detailed changelog.

    http://repo.or.cz/w/gpodder.git
    http://gpodder.org/news
    http://gpodder.org/downloads

Direct download link:

    http://download.berlios.de/gpodder/gpodder-2.6.tar.gz

Packages for Ubuntu, Maemo and Windows will be available shortly.


Thanks to all who contributed to make this release possible!

Enjoy!
Thomas
