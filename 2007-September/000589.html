Hello, gPodder translators!

Now is the right time to update your translations once again, as a new
version of gPodder will be released soon, and I'd like to ship updated
and complete translations with the package, if they are submitted in
time.

I've fetched the changed translations from Launchpad.net's Rosetta tool
and merged them with the current codebase. I've also updated the German
translation which should be complete now, and updated all translation
templates from the current source code.

If you want to translate gPodder using Rosetta, be sure to wait until
the translation template has been imported, you can see the status on
this page:

https://translations.launchpad.net/gpodder/trunk/+imports

If the translation has been imported, just head over to 

https://translations.launchpad.net/gpodder/trunk/

and start translating. Of couse, you can also do the translations in the
traditional way by grabbing the latest SVN trunk revision and using a
tool like "poedit" to translate your language file in "data/po/".

Shortly before the release, I will grab any updated translations from
the Rosetta tool and include it in the release. If you decide not to use
Rosetta, just send the updated translation to my e-mail address, and I
will make sure your translation gets into the new release.

Thanks for your help!

Thomas
