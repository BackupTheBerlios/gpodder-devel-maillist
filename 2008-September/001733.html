> What do you guys use for editing the data/gpodder.glade file?
> 
> The reason I ask is because glade-2 seems to have stopped working
> properly for me. If I load the glade file then save without any
> modification I end up with a 2200-line diff. I've tried on both the

I've been hand-editing the gladefile for the last few months, and
creating a new glade file and copying over the widget XML if I need to
design a new window. That way, the diff is always okay and I learn a
bit the structure of glade files. I believe that in the future, we
would have to migrate to gtk builder anyway, so depending on glade
isn't as good as it was some years ago.

The structure of .glade files isn't that hard to understand, although
it _does_ take a little bit of time getting used to editing them
without messing something up :)
