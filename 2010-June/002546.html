Hi, Frida!

On Tue, Jun 29, 2010 at 09:54:31AM -0700, Frida Karlsson wrote:
> My problem right now is that I just can't find the gpodder
> configuration file. When I look for the folder:
> /home/username/.config/gpodder/ it's just not there. Where can it be
> if that folder doesn't exist?

On Linux, it's under ~/.config/gpodder - you will find it in the hidden
folder ".config" in your home directory ("echo $HOME" in a terminal
should give you the location of your home directory).

On Windows, the configuration is stored in a subfolder of the directory
in which gpodder.exe resides (i.e. in the application folder). The
reason for this is that gPodder for Windows is thought as a "portable
app", so that users can put gPodder directly on a USB stick and use it.

HTH.

Thomas

PS: You can set the environment variables "GPODDER_HOME" and
"GPODDER_DOWNLOAD_DIR" to force using a different location - see the
section "Portable mode / roaming profiles" in the README file for
details.
