Hello!

I'm glad to announce the 0.11.2 release of gPodder, now available on the
website. This includes official support for the Maemo platform[1].

Thanks to all who sent patches, worked on features, sent a donation or
simply reported bugs. All your help is greatly appreciated and it's
really your work that really brings gPodder to a releaseable state every
month :)

Enjoy the weekend. Development will take on in early May when things at
University have settled down a bit. If you want to help out, please see
the Roadmap[2] for details on what we have planned for 0.11.3 and help
triaging bugs[3] and if you like, start working on one of the requested
features or simply leave some comments in the bug reports that might
help fix the issues reported.

Thanks,
Thomas

=============== release notes follow ==============

We're proud to release yet another gPodder version. This time, we call
it the "Walk like a Panther" release for no obvious reason, and we also
provide you with an official Maemo version of gPodder for all the
Internet Tablet fanboys out there.

Here's a list of the most important changes, for the full list of
changes, please see the ChangeLog file in the released tarball. We've
updated the list of contributors (patch writers, bug reporters and
donators) in the about dialog for this version. If you know you should
be on this list, but somehow aren't, please get in touch, and we'll add
you.

You can grab the release tarball now, get the Maemo version from the
Maemo repository (or use Application Manager if you already installed
gPodder from the repository on your Internet Tablet) or wait until your
distribution packages the new gPodder version for you :) Enjoy!

Thanks also to the people who sent donations during this release cycle -
your support is greatly appreciated! :)


User-visible changes (UI, features, ...)
      * Non-modal feed cache updates (Paul Elliot, Pieter De Decker,
        Jürgen Schinker and others)
      * Clean up preferences dialog + Move download settings to
        downloads tab (Paul Rudkin)
      * Change the context menu of the episode list popup menu (hex)
      * Additional fields for the device remove episodes dialog (iPod,
        MP3 player)
      * Report errors in parsing/loading an already-added channel in the
        navigator
Corrections, minor features and bug fixes
      * Add "Visit website" menu items to channel and episode list;
        simple chan menu
      * Correct icon for the "Transfer" toolbar icon and main menu item
      * Make bluetooth device discovery a bit nicer (don't destroy the
        button)
      * Correct "Close" and "Quit" behaviour (Harley Laue)
      * Merge patch from Jerome Chabod to fix tray icon freeze (bug #55)
      * Merge patch from Jerome Chabod to fix Remove podcasts for MP3
        players
      * Set Close and cancel buttons to the default on most dialogs
        (Jérôme Chabod)
Non-visible changes (Options, code compatibility)
      * Add option to limit the maximum number of episodes per feed
      * Add support for Rockbox' AudioScrobbler logfile to update played
        status (Nick)
      * Increase filename size limitation from 50 to 100 chars in MP3
        player sync (Bernd Schlapsi)
      * Make i18n help text work in the console help (gpodder --help)
        (Pavel Mlcoch)
      * Make sure we can re-set the locale after changing it
      * Make RSS .torrent feeds from different Torrent sites work (Josh
        Gardner)
      * Calculate total percentage based on byte size instead of
        percentage (Jérôme Chabod)
      * Make the Quit buttons actually quit when using the tray icon
        (Nick L.)
      * Fix bugs: install Makefile, TreeView header height
Technical code changes, management
      * Add demo code for displaying a splash screen (only in source,
        not in gPodder)
      * Timestamp and benchmark functions for logger
      * Delay loading of UserAppsReader and improve libplayers
      * Use cPickle for dumbshelve - it's compatible and faster
      * Make sure we ignore the updated_parsed of a channel if it's None
        (Chris Arnold)
      * Project management updates (authors, contributors and copyright)
Maemo-specific changes (gPodder Maemo)
      * Initial upstream support for the Maemo platform, i.e. Nokia
        Internet Tablets
      * Better tray icon menu item on Maemo-based platforms
      * Maemo finger-friendly updates; minor UI changes
      * Maemo-specific dialogs and UI improvements
      * Install "pixmaps" icon on Maemo, too
      * Add 64x64 icon for Maemo's main menu and install it
      * Fix problems with the tray icon emblems (for non-square icons)
Translations
      * Updated translations: Russian (Leonid Ponomarev), Spanish (Julio
        Acuña), French (Jérôme Chabod), Swedish (Anders Kvist), German
Donations
      * Mark Alford sent a donation - Thanks!
      * Sami Koykka sent a donation - Thanks!
