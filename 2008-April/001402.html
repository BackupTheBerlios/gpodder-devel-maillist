Hello, gPodder developers and users!

I've moved the roadmap entry for
http://bugs.gpodder.org/show_bug.cgi?id=82 from the 0.11.2 milestone to
the 0.11.3 milestone, because it means we are going to change strings
which in turn makes the translation get out of date. Because we are
already in feature freeze for 0.11.2, and we don't want to break the
strings again, I decided to move this change beyond the 0.11.2 release.

Apart from that, we're less than two weeks before the release now, and
there are still some things to do, although we've fixed most issues
already and Paul Rudkin finished the code for moving the download
settings to the downloads tab and improving the UI. I've made the
changes to the preferences dialog that I drafted in bug #83, which
should make the preferences dialog more straightforward to use.

The only things left before the 0.11.2 release are the
parallel/non-modal feed cache updates and part of the requested features
for the iPod sync module (see
http://bugs.gpodder.org/show_bug.cgi?id=56).

More information is available at http://wiki.gpodder.org/roadmap


Enjoy,
Thomas
