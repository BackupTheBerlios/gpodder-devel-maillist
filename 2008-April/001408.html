I'm currently working on one of the "big" changes for 0.11.2, which
should be out really soon now (we still have one week to finish things
up). I've added a patch (against SVN trunk) and a screenshot to show
what the results are to this bug report:

http://bugs.gpodder.org/show_bug.cgi?id=16

Please try out this patch, give some feedback and tell me if this is
better for you than the modal feed cache update dialog. I think the
place for the progress bar and the cancel button is in a good place now,
and we also see the progress when opening the main window after starting
the feed update from the tray icon's popup menu.

Also, cancelling the feed cache update now really works, in the code
with the dialog box, we didn't really cancel the downloads, but simply
hide the dialog (that was awful!). My fault ;)
