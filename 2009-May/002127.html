Hi Jerome!

It works under the new Ubuntu 9.04, but with some minor changes..

Getting the patch directly in one command fails because the path where
pymtp.py were used to, changes. So, this is that i did:

1. Locate where is pymtp.py

    locate pymtp.py

showing this result:

/usr/share/python-support/python-pymtp/pymtp.py
/var/lib/python-support/python2.6/pymtp.py

2. Backup original pymtp.py files (just in case)

sudo cp /usr/share/python-support/python-pymtp/pymtp.py
/usr/share/python-support/python-pymtp/pymtp.old.py
sudo cp /var/lib/python-support/python2.6/pymtp.py
/var/lib/python-support/python2.6/pymtp.old.py

3. Copy the patched pymtp.py (downloaded from here)
sudo cp ~/pymtp.py /usr/share/python-support/python-pymtp/
sudo cp ~/pymtp.py /var/lib/python-support/python2.6/

And now works! Thanks Jerome :)
