This should work (you can put these three lines into a text file named
"gpodder-windows.sh", then make it executable with "chmod +x" and then
just run it or even double-click it to start gPodder with these settings):

export GPODDER_HOME=/media/TI106049W0B/Users/simplicitasetmusica/Downloads/gpodder-2.16-win32/gpodder-2.16-win32/config/
export GPODDER_DOWNLOAD_DIR=/media/TI106049W0B/Users/simplicitasetmusica/Downloads/gpodder-2.16-win32/gpodder-2.16-win32/downloads/
gpodder -v

HTH :)
