A new release of gPodder is out, version 2.19 - another bugfix release
in our stable 2.x series of releases. The next one will be the first
release of the "tres" branch, so watch out for that one soon :)

Details here, see the notes below:
http://blog.gpodder.org/2011/09/gpodder-219-toast-derivation-released.html

Enjoy :)
Thomas

----------------------------------------------------------------

Before the big release of the "tres" branch as gPodder 3, we have
another bugfix release for the stable 2.x series of releases for you.
This release fixes various bugs found since the last release in August:

 * Fix empty descriptions in the Soundcloud module
 * Ignore image enclosures for audio/video in Media RSS (bug 1430)
 * Add Spanish and Turkish translations (bug 1420)

The detailed, full ChangeLog is available at

    http://gpodder.org/changelog/2.19

Download: http://download.berlios.de/gpodder/gpodder-2.19.tar.gz
md5sum: cad0cd6b303a38bab1c868b2186d816b
sha1sum: 6ac411241740fc93af165fdfcb355c9b02dcb714

Thanks to all who contributed to this release and reported
bugs :)
