Hi,

Based on Stefan Derkits' suggestion I've set up a GitHub organization
for gPodder and related projects [1]. This should make it easier for us
to keep track of others' developments and provide a central place for
people to discover gPodder related stuff.

I've imported/forked all relevant projects that came to my mind and have
assigned admin permissions to their maintainers (those should have
received an email from GitHub). If I forgot someone/something, please
let me know.

This does not mean, that any of those repositories should be considered
official. I'd like to start this as an experiment, and if anyone decides
to use the the central repository as their official one, please let us know.

Feedback on this idea is always welcome!


-- Stefan
