Hey Ari,

On 04/05/2011 10:04 AM, Ari Hannula (Nokia-MS/Oulu) wrote:
> Hi. How do I get all my subscriptions from gpodder.net as with:
> 
> http://gpodder.net/subscriptions/all.opml
> 
> Is there an API in libmygpo-qt? Couldn't find any.

you can do this via Device Updates ... if you want all Updates you can
always do it with Timestamp 0, but you should do it via incremental Updates.

libmygpo-qt doesn't (yet) support the Simple API of gpodder.net, so
either use the Advanced API Features or find some good Reasons to use
the Simple API and report a Bug for libmygpo-qt ;)
