Btw a really nice GIT Tip that is related to this (I will also put this
in the Wiki soon):

Add to your ~/.gitconfig the following Lines:

[url "git://github.com/gpodder/"]
        insteadOf = gpodder:
[url "git at github.com:gpodder/"]
        pushInsteadOf = gpodder:

Then you can clone the Repositories that are inside the gpodder
Organization by using "gpodder:repository.git", e.g. git clone
gpodder:libmygpo-qt.git

Or you can change the Remote to point to the short gpodder: Url

The Advantage of this Trick is to use the anonymous GIT to pull and only
use SSH for push, so that it doesn't need your SSH Password if you pull.
