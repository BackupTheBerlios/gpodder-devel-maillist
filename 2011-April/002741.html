Hello All

Today we have moved the gPodder Bug Tracker to a new server, the result of which is faster access to Bugzilla!  I also have used this as an opportunity to install various Bugzilla language packs so you can now access it in the following languages:-

	English
	German
	French
	Spanish
	Russian
	Japanese
	Traditional Chinese

It may take up to a day for the DNS entries to replicate world wide, but keep checking https://bugs.gpodder.org.

The server has been running a version of the bug tracker for the last couple of weeks so I don't anticipate any issues, however If you do notice anything strange then please let me know.
