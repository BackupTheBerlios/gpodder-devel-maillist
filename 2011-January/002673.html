For bug reports, I prefer bugs.maemo.org for the Maemo versions of
gPodder (N800, N810 and N900) and bugs.gpodder.org for the Desktop
version. I monitor both bug trackers. The reason why the Maemo versions
have a separate bug tracker is so that Maemo users don't need to create
a new account (if they already have a maemo.org account) to file bugs,
and so that gPodder's bugs there can depend on Maemo bugs directly. If
someone accidentally files a Maemo bug in bugs.gpodder.org, that's also
no problem.
