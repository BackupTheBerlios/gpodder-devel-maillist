Thanks for the nice bug report, I've checked the feed and found that
gPodder doesn't detect a channel to be selected when the item count in
the channel is zero. This has been fixed in current svn head trunk, so
run "svn up" in the root of your working copy, and "make test" should
reveal the missing menu items :)
