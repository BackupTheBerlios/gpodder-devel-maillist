However I got a problem running the program as the paths are hardcoded
in in gpodder. I use a setup like this for one of my programs (yet to
be published):

  def __init__(self):
    program_path = sys.path[0]
    lib_path = ""

    if os.path.exists(program_path + "/data/utfyll.glade"):
      lib_path = program_path + "/data/"
    elif os.path.exists(program_path+" /../share/utfyll/utfyll.glade"): 
      lib_path = program_path +"/../share/utfyll/" 
    else: 
      raise "Error: Cannot find the programs library files"

    glade_filename = lib_path + 'utfyll.glade'
    logo_filename = lib_path + 'utfyll-logo.png'
    icon_filename = lib_path + 'utfyll-icon.png'

Then the paths are more relative.
