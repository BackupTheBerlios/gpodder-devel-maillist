On Tue, 2007-03-27 at 18:07 -0400, Michel Salim wrote:
> One small question: why is there a src/gpodder.py.orig file?

This is needed for tepache to run. tepache automatically adds window
signals to the python code when added to the glade file. tepache uses
the .orig file to be able to compare it with the current gpodder.py to
check what it has to change to achieve the desired result.
