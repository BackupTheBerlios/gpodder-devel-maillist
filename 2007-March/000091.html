Hello!

I've just looked at the PyGTK docs to see if there is support for a
TrayIcon in it already. It seems[1] like there is support for the icon
in PyGTK >= 2.10.

[1] http://www.pygtk.org/docs/pygtk/class-gtkstatusicon.html

As PyGTK 2.10 does not seem to be widespread upon distributions (at
least the current stable releases), I suppose we should wait with the
implementation of a "real" tray icon with status display and menu.

Until then, alltray[2] lets you start gPodder in the system tray,
display its window title (download progress) as tooltip and makes it
even possible to minimize gPodder to the system tray with the "close"
button.

[2] http://alltray.sourceforge.net/


Enjoy.
Thomas
