Hi, Thomas!

I built this application mainly to play with a new language I'm
picking up, mobl (http://www.mobl-lang.org/). The application is
targeted at iPad as well as mobile devices. You can resize the browser
and it will adapt to a layout more appropriate for smart phones. This
is a built-in behavior for mobl applications :).  You can compile the
source code with mobl's command line tool, or Eclipse plugin, and the
compiled down code is all in www/ folder. You can host www/ on your
own server, or run it on top of fileserver.jar with command "java -jar
fileserver.jar 3000 www". There are some tutorials and screen casts
you can follow easily on mobl's website, and if you have any problem
running the app locally, feel free to drop me an email!

I don't have any big plan for it, so far it's just a proof of concept
thing. But I'm open to feature requests :).
