Hi,

I've just enabled IPv6-Support for the gpodder.net server. For testing
purposes, we currently use three different domains that all serve the
same service

 http://gpodder.net/         IPv4-only
 http://ipv46.gpodder.net/   Dual-Stack
 http://ipv6.gpodder.net/    IPv6-only

If you would like to test, please try to access all three sites. With
normal IPv4 internet access, you should be able to access the first two
only.

If you have dual stack internet access (or a IPv6 tunnel broker, etc)
you should, of course, be able to access all three sites.

If you can spend a bit more time on testing, you can also configure
gPodder to use one of the servers you can reach (change at Podcasts ->
Preferences -> Edit config -> Setting "mygpo_server").

Those that are extremely motivated can also get a IPv6 tunnel (I'd
recommend SixXS [1]) and try to access the IPv6-only site, too.

If you experience any problems, please let me know!


Thanks,

-- Stefan


[1] https://www.sixxs.net/
