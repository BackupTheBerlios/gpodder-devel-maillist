BTW, I don't know how rockbox presents the device in general, but with my
iAudio X5 running rockbox, it is indeed appearing as a USB disk to Linux and
gpodder.  This doesn't appear to be an attribute of rockbox though; it seems
to be doing synchronization using a part of the original firmware in the
device.  I have mine set up to dual boot rockbox and the Cowon firmware, but
the synchronization part seems unchanged.
