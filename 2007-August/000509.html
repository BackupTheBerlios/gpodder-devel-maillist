The trash symbol is there to visualize which episodes have already been
downloaded before. gPodder keeps track of which episodes have been
downloaded, so the "download new episodes" features works as expected.

Otherwise, if you would download the latest episode of a channel, listen
to it and later delete it, next time you click on "download new
episodes", the episode would be downloaded again.

With the trash icon, you can have a "download-play-delete" workflow of
listening to episodes while still being able to only download new,
unplayed episodes.

The trash-icon episodes will disappear from the list when the RSS feed
in question stops carrying the episode.

If you want to re-download a trash-icon episode, you can do so by
manually selecting it and clicking on "download" in the toolbar.
