Hello, gPodder users and devs!

I've worked on the experimental branch for quite a while now and after
lots of testing, I guess it's stable for developer consumption.

There's very cool, hot new stuff for you:
  - Better feed parsing
  - Faster user interface
  - Download cancelling possible from available podcasts tab
  - Context menu in available episodes list
  - Live download status in episode info dialog
  - Combined Status/Played column with composited icon
  - Channel list stored as OPML, local feed cache as shelve DB file
  - License upgraded to "GPLv3 or later" (from "GPLv2 or later")
  - Many cool bugfixes that make gPodder even more sexy!

Try it out, but don't forget to install the new dependency: feedparser.
On Debian and Ubuntu systems, feedparser can be installed with this:

  aptitude install python-feedparser

For all other distributions, check out your repositories or go directly
to www.feedparser.org and download from there.

Tell me if and how it works for you :)

Thomas
