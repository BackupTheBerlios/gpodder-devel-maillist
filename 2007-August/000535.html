On Fri, 2007-08-31 at 02:37 +0000, richardvoigt at gmail.com wrote:
> Will it pull in data from the old version?  Specifically the metadata
> for already-downloaded episodes that may no longer be listed in the
> online feed.

Yes, the localdb format hasn't changed, downloaded episodes are kept.
The only problem you could run in is if you modify your channel list and
then start 0.9.5 which will write (but not read) the channels.opml file.
In that case, simply adding the feed URL of the feeds that you added
after 0.9.5 will re-add the channel and also display the
already-downloaded episodes.

I plan to go from SAX to DOM/feedparser for reading/writing the localdb
channels, but the format will not change (it's basically an RSS file
with only the downloaded episodes), so already-downloaded episodes will
continue to work :)


Enjoy,
