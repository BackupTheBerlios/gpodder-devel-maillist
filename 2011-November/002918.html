Today I added two new hook scripts to implement this feature.

1. mp3gain
uses the command line program mp3gain. This tool don't decode and
re-encode the audio file but it could only handle mp3 format.

2. normalize_audio
uses the command line program normalize_mp3/normalize_ogg. This tool
decode and re-encode the audio file but could handle mp3 and ogg format.
If requested we could also add flac format, but at the moment the hook
only works for mp3 and ogg format.

https://github.com/gpodder/gpodder-hook-scripts/commit/3c8e52eec4d0e0614acccfaf44294ad928bf5aaa
