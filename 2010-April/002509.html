Hello!

This is the monthly update of the mygpoclient library, which is used by
gPodder to talk to the gpodder.net web service. This new release brings
some API updates (API Version 2, as deployed on the web service) and
will allow gPodder to sync more fine-grained playback episode actions to
the web service.

mygpoclient 1.3 will become a dependency of the next gPodder release,
and the Git version of gPodder will now depend on mygpoclient 1.3 to
function correctly. You can install mygpoclient 1.3 from the Ubuntu PPA,
from source or by checking out code directly from the Git repository and
creating a symlink to the "mygpoclient" package in "src/" in your
gPodder checkout.

Starting from this release, mygpoclient is also available from the
Python Cheese Shop: http://pypi.python.org/pypi/mygpoclient

The website with the official download link is located at:

http://thpinfo.com/2010/mygpoclient/


Enjoy!
Thomas
