On Tue, Mar 30, 2010 at 12:32:34PM -0300, Silvio Sisto wrote:
> I think I have a bug to report...
> 
> If you go to the view menu there are two similar options:
> 1: Show "All episodes" in podcast list
> 2: All episodes
> 
> If you go to preferences there's a third similar option:
> 3: Show "All episodes" in podcast list
> 
> I have the first option enabled in the view menu but the third option
> is disabled but apparently they are exactly the same option. Also I
> don't know why there seems to be 3 different places to set the same
> option.

Yes, this was my mistake (mechanically adding stuff from the Desktop
version to the Maemo 4 UI). The "1" and "3" are really the same, and
therefore I have removed the option from the main menu in the Maemo 4 UI
to avoid confusion. The "All episodes" item in the view menu controls
the view filter for the episode list, whereas the other menu item (now
in the preferences dialog) controls whether or not the item that
combines all episodes from all podcasts will be shown in the podcast
list.

Commit link here: http://gpodder.org/commit/3f6a2a1

Thanks,
Thomas
