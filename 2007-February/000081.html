Hi,
I think gPodder is a great tool for keeping up on the latest audio and video
podcasts, and I consider making it my default podcatcher. One major drawback is
that gPodder does not link itself into the tray. There are many good programs
that have certain features regarding the notification area or tray:

(1) it is possible to start the program minimized to the tray, so no window
shows up on the task bar, but the program is accessible via its tray icon

(2) the tray icon indicates the status (downloading, new podcasts available,
idle...)

(3) if the program window is open, a click on the window's close button in the
top right corner does not close the program totally, but resets it to the
"minimized in tray only" status described in (1)

Liferea does this all, and I really like to use it for my newsfeeds, but it
isn't made for podcasts. Please think about the propositions above, I feel
implementing them would make gPodder more attractive to a large group of people
(including me)...

Kind regards,
