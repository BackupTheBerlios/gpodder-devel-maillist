Hello!

I've just released gPodder 0.8.9, the release candidate for the
upcoming 0.9.0 release. Please download it from the gPodder homepage at 

http://gpodder.berlios.de/

For maintainers of gPodder packages in distributions: Please update
your package to the latest gPodder release and notify me about it so I
can update the gPodder downloads webpage accordingly.

Thanks to all who helped developing and testing gPodder in the last
months/weeks. Full release notes with credits on who did which feature
will be ready for the 0.9.0 release, which should be ready in some
days/weeks, depending on the feedback we get for 0.8.9.


Enjoy,
Thomas
