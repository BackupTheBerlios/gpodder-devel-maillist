attached patch provide some sync statistics functionality to gpodder,
use gpodder -S or gpodder --stats.
Maybe there's a place for it in gui too. Maybe there's a need to make
corresponding checks before each sync. New function sync_stats return
true or false depending on whether there's enough space on device to
sync all suitable episodes. Currently works with FS based devices
only.
