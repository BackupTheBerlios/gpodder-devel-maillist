Hello there!

Another release is out, thanks to all who were involved and helped
making this release the best gPodder release so far! We have added
some additional target systems for this release:

 * Maemo 5 (Fremantle)
 * Mer (Maemo Restructured)
 * Windows (XP/Vista)

This means that you can now run gPodder on more systems out there :)

Below you will find a copy of the announcement, as posted on

  http://gpodder.org/news

You can download packages for Debian, Ubuntu, Maemo 4 and Windows
and of course the source package on the downloads page at

  http://gpodder.org/downloads

A new set of screenshots has also been created to show 0.16.0 running
on different systems:

  http://gpodder.org/screenshots

Thomas

------------

After a productive weekend in Copenhagen with improvements in the UI for
portable devices we are ready to surprise you with another fine release
of the most enjoyable podcast client so far. Some highlights:

    * Download pausing, resuming and better error reporting
    * "Paste from clipboard" button in add subscription dialog
    * Auto-detect run path (makes --local obsolete)
    * Better screen space usage for the Maemo UI (no more tabs)
    * Newly supported target: Maemo 5 (Fremantle)
    * Newly supported target: Mer (Maemo Reconstructed)
    * Newly supported target: Windows (Tested with XP/Vista)
    * A HUGE set of bug fixes and performance improvements
    * Transition to GtkBuilder means removal of Glade dependency

See the detailed changelog for more information.
