1) I ran in to the problem of gpodder saying it "Cannot open device"
with a new iPod. The FAQ says the solution is to copy one file over using 
gtkpod.

But gtkpod looked complicated to me, and I already had iTunes on my Windows 
partition so I just rebooted in Windows and used iTunes to download and sync 
one podcast. Then I rebooted in Linux and gpodder worked fine.

I know that using iTunes like this is not a proper self-contained Linux 
solution, but it might be worthwhile to mention in the FAQ that this is one
option for some people.


2) The FAQ treats this question under the header
   'Why does gPodder report "can't access device"...'
But my version of gpodder says "Cannot open device", not "can't access
device".  Could the FAQ be updated to quote the correct error message?
